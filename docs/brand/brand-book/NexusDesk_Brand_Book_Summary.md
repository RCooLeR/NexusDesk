# NexusDesk Brand Book Summary

## Identity

**Name:** NexusDesk  
**Tagline:** AI Workbench for Code, Data & Ops

NexusDesk is positioned as a connected AI workspace for development, analytics, documents, marketing traffic, databases, and Docker/Ops.

## Visual direction

Option 3 is the selected direction: a futuristic orbital **N** mark. The orbit suggests a nexus of connected tools; the glowing nodes represent AI, code, data, documents, and operations.

## Logo variants

- Primary horizontal logo
- Stacked logo
- Orbital N symbol
- App icon
- Wordmark
- Monochrome symbol

## Color palette

- Deep Indigo — `#0B0F1F`
- Nexus Violet — `#7C3AED`
- Electric Blue — `#2563EB`
- Sky Blue — `#38BDF8`
- Silver — `#D1D5DB`
- Pure White — `#FFFFFF`

## Typography

Primary type family: **Inter**  
Recommended fallback: Arial / Helvetica / sans-serif

## Recommended first product screens

- Workbench dashboard
- Project/file explorer
- LLM chat panel
- Excel/data analysis view
- PDF/document viewer
- Docker/Ops panel
- Generated artifacts panel
